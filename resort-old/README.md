# 🏖️ Island Resort Website

Welcome to our Twin Island Resort project! 🌴 A luxury resort website built with Next.js, featuring a main resort island and a theme park island.

## ✨ Features

- 🏨 Resort booking system
- 🎢 Theme park activities
- 🚢 Inter-island ferry services
- 👥 User authentication
- 📱 Responsive design
- 💫 Modern UI with animations

## 🛠️ Tech Stack

- **Framework**
  - ⚛️ [Next.js 15.2](https://nextjs.org/)
  - 🌐 [React 19](https://react.dev/)
  
- **Styling & UI**
  - 🎨 [TailwindCSS](https://tailwindcss.com/)
  - 🧩 [Radix UI](https://www.radix-ui.com/) components
  - 📱 Responsive design with [clsx](https://github.com/lukeed/clsx)/[tailwind-merge](https://github.com/dcastil/tailwind-merge)
  
- **Database & Backend**
  - 🗃️ [Prisma ORM](https://www.prisma.io/)
  - 🔐 Authentication with [bcryptjs](https://github.com/dcodeIO/bcrypt.js)
  
- **Animation & Visualization**
  - ✨ [GSAP](https://greensock.com/gsap/) for animations
  - 🎭 [Framer Motion](https://www.framer.com/motion/) for transitions
  - 📊 [Recharts](https://recharts.org/) for data visualization
  
- **User Experience**
  - 🗓️ [Date-fns](https://date-fns.org/) for date handling
  - 💳 [React Credit Cards](https://github.com/amaroteam/react-credit-cards) for payment UI
  - 🔔 [React Toastify](https://fkhadra.github.io/react-toastify/) for notifications

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, generate the Prisma client:

```bash
npx prisma generate
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
